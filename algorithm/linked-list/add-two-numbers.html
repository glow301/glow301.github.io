    [leetcode-2] Add Two Numbers

    2021/3/16 LeetcodeLinked List算法

    # 题目大意

    给定两个非空的单链表代表两个非负整数，数字是逆序表示的，并且每个结点只包含一个单个数字。求两个数字的和，并用链表返回。

    每个结点不包含前导零，除非它自己是 0。

    # 思路

    这个题主要在于处理进位问题上。

    1. 如果两数之和大于 9，需要在下一次相加的时候带上进位。
    2. 如果一个链表已经遍历完成，另一链表的下一位加上进位大于 9，需要继续考虑进位。
    3. 两个链表都遍历完成，如果有进位，需要额外增加一位。

    # 代码

    /**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func addTwoNumbers(l1 *ListNode, l2 *ListNode) *ListNode {
    head := &ListNode{}   
    node := head
    sum := 0
    for l1 != nil || l2 != nil {
        if l1 != nil {
            sum += l1.Val
            l1 = l1.Next
        }
        if l2 != nil {
            sum += l2.Val
            l2 = l2.Next
        }
        node.Next = &ListNode{sum%10, nil}
        node = node.Next
        sum = sum/10
    }
    if sum != 0 {
        node.Next = &ListNode{sum, nil}
    }
    return head.Next
}
